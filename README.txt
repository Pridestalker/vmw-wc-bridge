This plugin offers a bridge between the 'Vindmijnwijn verkopers portaal' and your WooCommerce webshop.

== Description ==

This plugin aims to products from your WooCommerce site with the custom portal of Vindmijnwijn.nl.


== Installation ==

1. Upload `vmw-wc` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Follow the installation steps
